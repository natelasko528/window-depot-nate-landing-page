# V3 GoHighLevel Posting Plan (Pre-Publish Review)

## Step-by-step execution plan before sending to GHL
1. Lock creatives: use only `ad-drafts/30-posts/branded-v3/post_XX_branded_v3.png` (no image edits).
2. Approve platform copy per post in `ad-drafts/30-posts/v3_platform_post_copy.json`.
3. In GoHighLevel > RevolutionAI Marketing Group > Social Planner, confirm Facebook, Instagram, and LinkedIn pages are connected.
4. Add destination links with UTM template: `utm_source={{platform}}&utm_medium=social&utm_campaign=wdusa_v3_photo_first&utm_content=post_XX`.
5. Apply staggered schedule from `ad-drafts/30-posts/v3_ghl_staggered_schedule.csv` (America/Chicago).
6. Run QA for each queued post: platform copy match, image match, phone number, offer stack, and URL.
7. Pilot release: first 3 posts only; review CTR, comments, saves, and click-through after 48 hours.
8. Full release: publish remaining posts, then optimize weekly by promoting top performers and revising low performers.

## Platform conversion methods used
- Facebook: PAS + social proof + offer stack + low-friction CTA; no hashtags.
- Instagram: hook + image angle + benefit bullets + CTA + 10-12 hashtags.
- LinkedIn: insight-led framing + homeowner impact + credibility + soft CTA; max 3 hashtags.
